About SAMPARK

Need a Secure encryption GUI? I can Help You!

Sampark is an Image to Audio inter-converter. The data of vital importance is encrypted into an audio file. Then the dncrypted data can only be decoded via SAMPARK decoder. This allows wireless and wired data transmission in form of an audio or image file.
